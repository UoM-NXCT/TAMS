Welcome to the TAMS documentation
=================================

TAMS is an open-source tomography archival and management application released by
`National X-ray Computed Tomography (NXCT) <https://nxct.ac.uk/>`_, the UK's national
lab-based X-ray computed tomography research facility.

It aims to make storing and accessing data easy and quick.

TAMS is open-source and released under the `MIT licence
<https://github.com/UoM-NXCT/TAMS/blob/main/LICENCE>`_.
